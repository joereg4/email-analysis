🎯 What This System Does

  • 🦠 ClamAV Scanning: Antivirus detection for malware
  • 🔍 YARA Rules: Custom pattern matching for threats
  • 📷 EXIF Analysis: Image metadata extraction
  • 👁️ OCR Processing: Text extraction from images/PDFs
  • 🌐 URL Analysis: VirusTotal integration for link checking
  • 🤖 AI Analysis: OpenAI-powered threat assessment
  • 🚫 Quarantine: Automatic isolation of suspicious content

🚀 Getting Started

  1. Configure API Keys: Add your OpenAI and VirusTotal API keys to the .env file
  2. Start Services: Run ./setup.sh to start all Docker services
  3. Upload Emails: Drop .eml files into the inbox folder or use the web interface
  4. View Results: Check the dashboard for analysis results and risk scores
